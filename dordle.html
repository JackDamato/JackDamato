Dordle
WORDLE DORDLE NERDLE

Dordle functions exactly like wordle, except for the fact that there are two words that you have to guess using the same 7 guesses for each word! If a letter is green, that means it is in the right position, if a letter is yellow it is in the wrong position and if it remains white it is not in the word. Good Luck...
